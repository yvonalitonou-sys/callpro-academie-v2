# CallPro Académie

Application de formation au métier de téléconseiller / conseiller client. 12 modules couvrant la communication professionnelle, la structure d'appel CROC, l'argumentation CAB, le traitement d'objections (CRAC), la gestion de conflits (DESC), un simulateur CRM avec plusieurs mises en situation, et la préparation à l'entretien d'embauche (25 questions modèles).
